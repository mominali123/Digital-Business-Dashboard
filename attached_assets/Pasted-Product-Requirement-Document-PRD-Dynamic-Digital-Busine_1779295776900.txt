Product Requirement Document (PRD): Dynamic Digital Business Card Platform1. Product Overview & ObjectivesThe goal is to build a web-based SaaS platform that allows professionals, freelancers, and businesses to create, host, and share premium, interactive digital business cards based on the aesthetics of the provided source code.Key ObjectivesAccessibility: Allow users to spin up a professional card in under 5 minutes without writing code.Aesthetic Continuity: Retain the signature modern card aesthetic: ultra-clean typography, smooth hover animations, micro-interactions, and a context-aware dynamic SVG background.Portability: Offer both hosted links (e.g., card.bio/username) and an option to export a standalone, self-contained HTML file (just like your template).2. Core User FlowOnboarding/Sign-up: User creates an account.Details Input: User fills out fields (Name, Title, Company, Bio, Links).Design Customization: User selects a primary accent color, typography style, and an "Industry Icon Pack" for the background canvas.Publish/Export: Card is published to a live URL, and a custom QR code is generated for quick real-world scanning.3. Functional Requirements3.1. Profile Data Management (The Content)The system must provide an intuitive form builder to accept the following fields:Brand Header: Brand Name (lowercase styling option) and Subtitle/Slogan.Personal Info: Full Name, Professional Title, and Location (with toggle for a live status dot).Bio/Tagline: Short italicized text block (maximum 150 characters).Action Links: Structured inputs for common action buttons:Communication: WhatsApp, Phone Call, SMS, Email.Socials: LinkedIn, Facebook, Instagram, X (Twitter), YouTube, GitHub.Custom Links: Custom URL with an emoji or custom selectable icon.3.2. Design & Layout Engine (The Visuals)Instead of hardcoded CSS variables, the platform will inject user choices into the component stylesheet:Theme Engine: Ability to change --red (Primary Accent), --ink (Text Color), and --off-white (Background Fill).Logo Customizer: Option to upload a circular profile image/logo inside the animated pulsing ring or use a pure CSS text abbreviation.Staggered Animation: Maintain the staggered entry animation (btnIn) for call-to-action buttons upon loading.3.3. Dynamic SVG Background Scatter GeneratorOne of the best features of your file is the background JavaScript that randomly places icons without collisions. The platform will turn this into a killer feature:Icon Pack Selection: Users pick an industry sector which swaps the icons array in the background script.Sourcing/Tech (Current): Laptop, Camera, Battery, Smartwatch, TV, Truck.Real Estate: House, Key, Building, Blueprint, Location Pin.Creative/Design: Pen, Brush, Palette, Layer-group, Monitor.Medical/Health: Heart, Stethoscope, Pill, Clipboard, Cross.Scatter Settings: Expose control over background icon density (count multiplier), opacity ranges, and rotation angles via UI sliders.3.4. Sharing & DistributionQR Code Generator: Every published card automatically generates a downloadable, high-res QR code mapping directly to the card's URL.Single-File HTML Export: A "Premium Feature" allowing users to download a compiled, minified .html file with all CSS, SVGs, and background scripts embedded, letting them host it completely independently.4. UI/UX & Design GuidelinesTo match the pristine quality of your template, the generated output must strictly follow these UI guardrails:ElementSpecification / BehaviorCard ConstraintsMax width capped tightly at 380px on desktop; fully fluid (92vw) on mobile displays.TypographyPremium font pairings (e.g., Outfit for tech/modern, Playfair Display for luxury, Inter for corporate).Button InteractionsHovering over any action link triggers a smooth transition that slides padding inwards (padding-left: 38px) and coaxes the right-hand chevron forward (translateX(3px)).Pulsing AccentsThe logo ring must loop a gentle glowing pulse effect using CSS keyframes (ringPulse) to draw attention without being distracting.
